Simple WooCommerce button text changer / Styler / add to cart text changer for WooCommerce.

== Description ==
Woo Button Text Changer can change woocommerce button text globally. You can change button name from WordPress admin panel. and also change the color, shape , and styles also special of related product remove from single product page.

== Installation ==

1. Upload the entire 'woo-button-text' folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Woo Button Text' menu in your WordPress admin panel under Settings menu.

For basic usage, you can also have a look at the http://www.exclutips.com/plugins/woo-button-text/
